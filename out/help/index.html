QuickIRC is designed to be simple to use without a lot of noisy bells and whistles. If you just want to connect, this IRC client is for you!

But maybe you are looking for something feature-rich and complicated. In that case, XChat and HexChat are both very good alternatives. QuickIRC's strength is in its simplicity.

Getting Started

When QuickIRC first opens, you'll see the connection tab. Fill in the host and nickname. The username will automatically fill in. Hit Connect!

Once the connection comes up, type in /join #chat (replacing #chat with the channel you want to join). Hit Enter, and the channel chatroom will appear!

Commands

QuickIRC aims to support all standard IRC commands, with the exception of some client-to-client (CTCP) functionality. The most common commands are listed below.

Global Commands

These commands are valid in any context.

/join <channel_name>
Joins the specified channel. You can join more than one by separating them with commas.
/msg <nickname>
Sends a private message to another user.
/nick <new_nickname>
Changes your nickname that is visible to others.
/part <channel_name> [reason]
Leaves the specified channel. Optionally, you can specify a parting message.

Channel Commands

These commands and syntaxes are available within channel chats.

/part [reason]
Leaves the current channel. Optionally, you can specify a parting message.
/say <message>
The same as sending a message like normal, except now it can start with a forward-slash!
/topic
Shows the current topic of the channel in chat.
/topic <new_topic>
Changes the topic of the channel. Note, you must have permission to change the topic for this to work.

Private Message Commands

These commands and syntaxes are available within private message chats.

/part
Closes the private message chat.
/say <message>
The same as saying a message like normal, except now you can start your message with a forward-slash!
